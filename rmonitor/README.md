# Mantra Redis Server 


## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server

```bash
node server.js
```

## Production

Build the application for production:

```bash
node server.js
```

Locally preview production build:

```bash
node server.js
```
